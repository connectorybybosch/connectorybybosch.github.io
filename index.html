¿Qué es el XDK 110?

Es un disposivo desarrollado por Bosch Connected Devices and Solutions para crear productos IoT, capaz de cumplir cualquier necesidad de prototipado; cuenta con 6 sensores programables:

  • Sensor geomagnético de tres ejes (BMM150)
  • Sensor de aceleración de tres ejes (BMA280)
  • Sensor de velocidad angular de tres ejes (BMG160)
  • Sensor ambiental (BME280)
  • Sensor de ruido acústico (AKU340)
  • Sensor de luz ambiental (MAX44009)
Ejemplos de uso

Conectividad

Hay diferentes formas en las que es posible interactuar con el XDK. Aquí los protocolos

HTTP y HTTPS

El XDK soporta HTTP nativamente y ofrece dos módulos para hacer peticiones

Bluetooth

El XDK proporiciona diferentes APIs BLE para gestionar la funcionalidad Bluetooth.

LWM2M

Protocolo M2M que reduce la transferencia de datos innecesarios como http cuando se conectan dispositivos embebidos

USB

Interfaz para todos los dispositivos periféricos, que se utiliza para transmitir datos y fuente de alimentación

MQTT

Protocolo de mensajes diseñado para comunicación M2M y aplicaciones Internet of Things

CoAP

Constrained Application Protocol (CoAP) diseñado para ser barato y para trabajar con microcontroladores con poca RAM
